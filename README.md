# node-ci-practice

This is a backend project built with **Node.js**.

The project is created for learning and practice purposes.  
It uses **GitHub Actions (CI)** to automatically check the code whenever changes are pushed to the repository.

## Features
- Node.js backend structure
- Automated CI workflow using GitHub Actions
- Clean and simple project setup

## Purpose
This project helps me practice:
- Backend development
- Git and GitHub
- Continuous Integration (CI)

More features will be added step by step.
